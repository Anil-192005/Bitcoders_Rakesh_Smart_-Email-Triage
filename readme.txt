project title: AI investor advice

short description of the project: This project is a website that uses AI to check company details and suggest the best investment option. Admins can edit data, and normal users can get AI recommendations.

problem statement : ai based investor sentiment analyzer for m&A--predict reactions to mergers ; track sentiment+success likelihood

features : 
->Login System with Roles
->Role-Based Dashboard
->Live Company Investment Database
->Investment History Table
->AI Recommendation System
->User Investment Actions
->Responsive & Styled UI
->Dynamic UI Updates

Tech stack used:
Frontend Structure: HTML
Styling / UI Design: CSS
Frontend Logic / AI Scoring : JavaScript
Data Storage : Local JavaScript Objects
Platform : Web browser
Backend Structure: Python

set up & how to run : 

step1: download the project file and extract the folder
step2: Open the folder in VS Code
step3: Click “Open with Live Server” or open "index.html" in browser
